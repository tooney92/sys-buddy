# sys-buddy v1.0.2 — 2026-07-23

Infrastructure release. No agent-visible or tool-contract changes.

## Added
- **Docker packaging.** A multi-stage root `Dockerfile` (built with `uv`) and
  `docker-compose.yml` run the broker as a single container, persisting the SQLite DB
  in a `sys-buddy-data` volume.

## Notes
- Packaging/dev-tooling only; the broker's MCP tools, wire contract, and dashboard
  behavior are unchanged from v1.0.1.
